IP Addressing Explained: IPv4, IPv6, CIDR & IP Types

Complete IP Addressing Guide for Networking and Cybersecurity Beginners

What is an IP Address?

An IP address is a unique logical address assigned to a device connected to a network.

IP stands for Internet Protocol. It allows devices to identify and communicate with each other over a network.

Just as a house needs a postal address to receive letters, a device needs an IP address to send and receive network data.

Example of an IP Address

192.168.1.10

When one device communicates with another device, IP addresses help identify the source and destination of network traffic.

Why are IP Addresses Important?

IP addresses are essential for communication between devices on a network.

  • Identify devices on a network
  • Allow communication between devices
  • Help routers forward packets
  • Identify source and destination devices
  • Help network administrators manage networks
  • Help security professionals analyze network traffic

What is IPv4?

IPv4 stands for Internet Protocol Version 4.

IPv4 uses a 32-bit address. It is divided into four 8-bit sections called octets.

IPv4 Example

192.168.1.10

Each octet can contain a value from 0 to 255.

IPv4 Address Structure

192 . 168 . 1 . 10

  • 192 = First Octet
  • 168 = Second Octet
  • 1 = Third Octet
  • 10 = Fourth Octet

What is IPv6?

IPv6 stands for Internet Protocol Version 6.

IPv6 was developed because the number of available IPv4 addresses is limited.

IPv6 uses a 128-bit address.

IPv6 Example

2001:0db8:85a3:0000:0000:8a2e:0370:7334

IPv6 addresses use hexadecimal numbers separated by colons.

IPv4 vs IPv6

IPv4 IPv6
32-bit address 128-bit address
Example: 192.168.1.1 Example: 2001:db8::1
Decimal format Hexadecimal format
Limited address space Very large address space

What is a Public IP Address?

A Public IP address is an IP address that can be reachable over the Internet.

Internet Service Providers assign public IP addresses to homes, businesses and organizations.

Public IP addresses must be globally unique.

What is a Private IP Address?

A Private IP address is used inside local networks such as homes, offices and organizations.

Private IP addresses are not directly routable over the public Internet.

Private IPv4 Address Ranges

Range CIDR
10.0.0.0 - 10.255.255.255 10.0.0.0/8
172.16.0.0 - 172.31.255.255 172.16.0.0/12
192.168.0.0 - 192.168.255.255 192.168.0.0/16

What is a Static IP Address?

A Static IP address remains fixed and does not normally change automatically.

  • Servers
  • Network devices
  • CCTV systems
  • Web hosting
  • Remote access systems

What is a Dynamic IP Address?

A Dynamic IP address is automatically assigned to a device.

DHCP is commonly used to automatically assign IP addresses to devices.

Dynamic IP addresses may change over time.

Network ID and Host ID

An IP address contains two important parts:

  • Network ID: Identifies the network.
  • Host ID: Identifies a device inside the network.

What is a Subnet Mask?

A subnet mask is used to separate the network portion and host portion of an IP address.

Example: 192.168.1.10

Subnet Mask: 255.255.255.0

This commonly represents the network: 192.168.1.0/24

What is CIDR?

CIDR stands for Classless Inter-Domain Routing.

CIDR notation represents the number of network bits in an IP address.

Example: 192.168.1.0/24

The number /24 means that the first 24 bits represent the network portion.

CIDR is very important for subnetting and efficient IP address allocation.

IPv4 Address Classes

Traditional IPv4 addressing divided IP addresses into different classes.

Class Range Usage
Class A 1.0.0.0 - 126.255.255.255 Very large networks
Class B 128.0.0.0 - 191.255.255.255 Medium-sized networks
Class C 192.0.0.0 - 223.255.255.255 Small networks
Class D 224.0.0.0 - 239.255.255.255 Multicast
Class E 240.0.0.0 - 255.255.255.255 Experimental

Modern networks primarily use CIDR instead of traditional class-based addressing.

Special IP Addresses

Loopback Address

The IPv4 loopback address is: 127.0.0.1

It is commonly known as: localhost

APIPA Address

APIPA stands for Automatic Private IP Addressing.

If a device cannot receive an IP address from a DHCP server, it may automatically assign itself an address in the range:

169.254.0.0 - 169.254.255.255

Types of IP Communication

Unicast

One sender communicates with one receiver.

Broadcast

One sender communicates with all devices on the local network.

Multicast

One sender communicates with a specific group of receivers.

Anycast

Data is sent to the nearest or best available device from a group of devices.

What is a Default Gateway?

A Default Gateway is the device that forwards traffic from a local network to another network.

In most home networks, the router acts as the default gateway.

Example: 192.168.1.1

IP Address and DNS

DNS translates domain names into IP addresses.

Example: example.com → IP Address

What is NAT?

NAT stands for Network Address Translation.

NAT allows multiple private devices to share a public IP address when accessing the Internet.

IP Addressing and Cybersecurity

Understanding IP addressing is extremely important for cybersecurity professionals.

  • IP address scanning
  • IP spoofing
  • Network reconnaissance
  • Firewall rules
  • Access Control Lists
  • Network monitoring
  • Intrusion detection

IP Address Troubleshooting

  • Incorrect IP address
  • Incorrect subnet mask
  • Incorrect default gateway
  • DHCP server failure
  • IP address conflict
  • DNS configuration problems

Useful IP Address Commands

Windows: ipconfig

Linux: ip addr

Conclusion

IP addressing is one of the most important fundamentals of computer networking.

A strong understanding of IPv4, IPv6, public and private IP addresses, subnet masks, CIDR, gateways and IP communication types is essential for networking and cybersecurity.
